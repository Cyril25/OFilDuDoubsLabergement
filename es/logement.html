El alojamiento

Un espacio cálido para sentirse como en casa

Descubra nuestro encantador alojamiento, diseñado para ofrecerle una estancia inolvidable en Labergement-Sainte-Marie. Cada habitación ha sido pensada para combinar confort, modernidad y convivencia, satisfaciendo todas sus necesidades. El alojamiento tiene capacidad para 4 personas, ideal para estancias en familia o con amigos.

El salón y el comedor

Un espacio luminoso y acogedor, perfecto para reunirse en familia o con amigos. Dispone de una televisión de pantalla grande para sus noches de cine, una pequeña mesa de centro, una mesa de comedor para 4 personas y una selección de juegos de mesa para momentos de compartir y relajarse. Un espacio interior cálido, ideal para recargar energías después de un día de actividades.

La cocina

Una cocina moderna y totalmente equipada, ideal para cocinar con total libertad durante su estancia. Incluye horno, microondas, lavavajillas, placa de cocina, nevera con congelador, cafetera de filtro y máquina Nespresso, tostadora, así como toda la vajilla y utensilios necesarios. Para mayor comodidad, también dispone de un aparato Seb EasyFry, una máquina de raclette y un juego de fondue, perfecto para comidas agradables después de un día ajetreado.

El dormitorio principal

Una habitación espaciosa con una cómoda cama doble, decoración elegante y todo lo necesario para una estancia agradable. Cuenta con un gran armario, una cómoda para facilitar el almacenamiento y una televisión. Mesitas de noche con lámparas se encuentran a cada lado de la cama para mayor comodidad. Esta habitación también tiene acceso directo al balcón para disfrutar del exterior.

El dormitorio secundario

Una habitación acogedora, equipada con una cama doble, un armario y mesitas de noche con lámparas para dos. Ideal para noches de descanso, en un entorno tranquilo y cálido. Esta habitación también tiene acceso directo al balcón para disfrutar del exterior.

El baño

Un baño moderno equipado con una bañera para un momento de relajación. La iluminación, regulable en intensidad y color, crea un ambiente relajante. Encontrará toallas suaves, artículos de aseo como champú y gel de ducha, así como un secador de pelo y una lavadora para mayor comodidad.

El balcón

El exterior ofrece un espacio agradable con un salón cómodo, perfecto para un aperitivo al aire libre mientras disfruta de una vista excepcional. Una mesa y sillas están disponibles para comer 4 personas, y una parrilla eléctrica está a su disposición para realizar deliciosas barbacoas. Un lugar ideal para relajarse y disfrutar de los días soleados.
