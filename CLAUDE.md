# O'Fil du Doubs — instructions Claude Code

Site vitrine statique du gîte (GitHub Pages, https://ofildudoubs.fr), multilingue,
avec crons GitHub Actions qui rafraîchissent `data/` (POI, agenda, randos) et
workers Cloudflare dans `workers/`.

## Suivi & réflexion (repo notes séparé)

La fiche de suivi de ce projet est dans `..\notes\ofildudoubs.md` (repo git séparé, cloné à côté de celui-ci).

- **Début de session** : faire un `git pull` dans le repo notes puis lire la fiche (journal, à faire, réflexions).
- **Fin de session** : y reporter ce qui a été fait (entrée datée dans le Journal), les frictions et idées, puis commit-push le repo notes.
